# shenzhen-io-lang

This is a WIP interpreter for the pseudo-assembly language featured in the game
Shenzhen I/O.


| Implemented |
| ----------- |
| mov         |
| add         |
| mul         |
| div         |
| sub         |
| not         |
| jmp         |
| teq         |
| tgt         |
| tlt         |

For executing, the instructions can take in one or many of these arguments.

```
R - Register
L - Label
I - Integer
```
